# Clever_LendCo

Modernized digital menu artifact for **The Brack Shack Bar & Grill**.

## Files
- `menu.html` — responsive menu layout with an embedded QR code block for quick phone access.

## Preview locally
```bash
python3 -m http.server 4173
```
Then open `http://localhost:4173/menu.html`.

> Update the `qrTarget` variable near the bottom of `menu.html` to your final hosted menu URL.
